# Bundled fonts — licensing

These fonts are self-hosted (shipped as module assets) so the backend never
calls a font CDN at runtime.

| File | Family / weight | Licence |
|------|-----------------|---------|
| `montserrat-600/700/800.woff2` | Montserrat (SemiBold/Bold/ExtraBold) | SIL Open Font License 1.1 — © Julieta Ulanovsky et al. |
| `opensans-400/600/700.woff2` | Open Sans (Regular/SemiBold/Bold) | SIL Open Font License 1.1 — © Steve Matteson / Google |

Latin-subset `woff2` files, sourced from the Fontsource distribution
(`@fontsource/montserrat`, `@fontsource/open-sans`). Both families are under the
SIL OFL 1.1, which permits bundling and redistribution with the software.
